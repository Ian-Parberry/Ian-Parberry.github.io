ASCII Art Image Index

The table below gives a list of links to my favorite ASCII art pieces, with a brief description of their properties. This list currently has 110 ASCII Art pieces made by Ian Parberry from 26 different images.

Under the "Colors" column, 0 means that the file is text (the remainder are png images), 1 means that it is monochrome, 3 means that it is 3-color (CMY), 4 that it is 4-color (CMY + black), and 16M that it is full 24-bit color (see my color ASCII art webpage if you are unsure what this means). The next two columns give the width and height in characters. The approximate image width and height in pixels can be obtained by multiplying these values by 8. The last two columns give credit for the creator of the original image and the place in which it was created.

See my ASCII art webpage for more information on how these ASCII art pieces were generated.
