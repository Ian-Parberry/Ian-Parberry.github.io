Pseudo-Random Bits from Finite Groups

Ian Parberry, "Pseudo-Random Bits from Finite Groups", 2020.

Abstract

A sequence of pseudo-random permutations is computed using a pair of generators that with high probability generate either the symmetric group or the alternating group. A pseudo-random bit stream computed from this sequence appears to perform well on the Dieharder test battery.

Supplementary Material

Cross-platform C++ code is for the Cayley pseudo-random number generator is available from https://github.com/Ian-Parberry/cayley.git. For instructions on how to compile, run, and grok the code, read the Doxygen-generated documentation. The Dieharder output files used in the paper are available as a 400KB zip file.
