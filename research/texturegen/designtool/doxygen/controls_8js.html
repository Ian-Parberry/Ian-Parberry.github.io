Detailed Description

Functions that help manage HTML controls.

These include functions to manage the edit boxes that contain the coefficients a0, ..., f0 and a1, ..., f1; and radio button handlers that get called when radio buttons are clicked.

function AdvancedRadioButtonHandler ( )

Advanced radio button handler.

Handler for when the advanced style radio button is clicked. Enable the direction radio buttons. Call the handler for whichever one is checked so that the correct group(s) of coefficients get enabled.
