Colorize a grayscale value.

Given a grayscale value and the number of color bands, see which band the greyscale value is in, grab the color assigned to that band, and scale it by the normalized grayscale value. Oh, and first invert the grayscale value if the "invert" checkbox is checked.

Parameters
vGrayscale value between 0.0 and 1.0.
nNumber of colors, must be at least 1.
Returns
An array of floating point RGB values between 0 and 255.

Compute a modular bivariate quadratic function.

This is the key to the whole project. We're essentially attempting to hash 2D integer coordinates and failing in an interesting way. The hash function used is (ax^2 + by^2 + c xy +dx + ey + f) mod q. This function returns that value divided by q-1 to bring the result between 0 and 1. The code should actually be quite straightforward, but it's not because JavaScript.

Parameters
xcoordinate.
ycoordinate.
Returns
Hash of x and y, between 0.0 and 1.0.
