Detailed Description

Functions that build and parse a URL containing biq parameters.

Biq will allow URL parameters that specify the settings for a particular texture so that interesting ones can be saved as a URL. This involves code to build such a URL, which is displayed so that the user can right-click and copy it, and code to parse such a URL.
