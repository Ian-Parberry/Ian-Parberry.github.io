Publications by Keyword

Ian Parberry is author or coauthor of over 100 academic publications that include six books and over 75 articles (23 as sole author) in peer-reviewed journals, conferences, and workshops. These cover a wide range of computing topics that include analog computing [24] [25] [26] [27], ASCII art [58], artificial neural networks [6] [24] [25] [26] [27] [37] [41] [43] [48] [49] [50] [66] [67] [68] [69] [82], audio computing [76] [77], circuit complexity [6] [16] [26] [31] [43] [49] [66] [67] [68] [69] [84], combinatorial search [34] [39] [62], computational complexity theory [32] [37] [48] [49] (see also circuit complexity), computational economics [11], computational geography [60] [61], computational learning theory [25] [27] [41], computational meteorology [20] [73], computational number theory [79], computer graphics [1] [2] [3] [4] [20] [59] [60] [61] [72] [73] [74] [85], computer science education [14] [15] [46]  (see also game programming education), data structures [45], design and analysis of algorithms [5] [34] [39] [47] [51] [59] [62], design automation [17], entertainment computing  (see game programming, video game cognitive assessment), fault-tolerant computing [6], game programming [76] [77]  (see also procedural content generation, game programming education), game programming education [12] [13] [44] [53] [54] [55] [56] [57] [63] [64] [65], interconnection networks [29], load sharing [45], mathematical puzzles [19] [47] [50] [51] [62], mechatronics [18], packet routing [36], parallel computing [28] [29] [30] [31] [32] [33] [35] [36] [45] [79] [66] [68] [78] [83], path planning [5] [8] [17] [18], procedural content generation [1] [2] [3] [4] [9] [10] [11] [59] [60] [61] [71] [73] [75] [80] [81] [85], philosophy of computing [52], randomized algorithms [67] [69], real-time algorithms [47] [62], robotics [18], software agents [10], sorting networks [34] [38] [39] [40] [42] [70], video game cognitive assessment [21] [22] [23] [81], and visualization of algorithms [7].
